# CouchTV Fire TV App

Native Android TV app for Amazon Fire TV Stick.

## Stack

- Kotlin
- Jetpack Compose (TV-first UI in upcoming chunks)
- Media3 ExoPlayer
- Retrofit + OkHttp

## Current chunk status

- Base Android app scaffold
- Leanback launcher manifest setup
- Minimal Compose screen
